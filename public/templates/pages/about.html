About

What about Free Video Player? The idea is to create a javascript HTML5 video player that is able to play adaptive bitrate streams aswell as regular video boxes, such as mp4 etc. The open source license is GPLv3, see http://www.gnu.org/licenses/gpl-3.0.en.html for more information.

There are a number of different video players based on HTML5 technologies both commercial versions (closed source) and some open source, but most of (if not all) video players claiming to use open source are only using a more narrow license (Apache) which in turn does not provide full transparency and ensuring that the actual code will remain "free" to the end user as described in http://www.gnu.org/philosophy/philosophy.html. For more information about free software check out the Free Software Foundation. There are other video players and also open source video players, but none using the GPLv3 license so far I know of. The project is available @ GitHuba For more information about the API and methods used check Free Video Player Docs.

The main idea is to utilize the Source Media Extension module when available and in the future the Encrypted Media Extension module to build the player methodologies around. Check out the Installation page to get more info about how to install the Free Video Player.
