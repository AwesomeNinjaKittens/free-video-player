Api

There are a number of publically available methods through the Free Video Player API. Below these are described. The latest changes to the code (and also the API) can be found in the documentation section. Basically lets imagine we instantiate a Free Video Player, such as var player = freeVideoPlayer(optionalConfigurationObject); then we can access the publically methods as described in the table below.

Method Description
player.load(videoUrl, optionalConfigurationObject) - Loads the video into the player, should provide a url to where the asset can be found. The optionalConfigurationObject contains information such as alternative splash image and such.
player.play() - Starts playback of the currently loaded asset
player.pause() - Pauses playback of the currently loaded and playing asset.
player.getVolume() - This method returns the video element volume as a number between 0 and 1.
player.setVolume(numberFromZeroToOne) - This method interacts with the player video element and sets the volume within the stream/media. the variable numberFromZeroToOne should be a decimal number from zero to one, such as 0.8 for instance.
player.seek(positionInSeconds) - This method interacts with the player video element and seeks within the stream/media. The variable positionInSeconds is the position since start, we want the player to skip to, should be an integer from 0 and up.
player.fullscreen() - This method interacts with the player video element generates or exists a fullscreen mode.
player.getArrayOfSubtitleObjects() - A return method that could be public to return the current video's subtitle info in an array of subtitleObjects containing values such as label, language, id. If a user/developer do not want to use the built in custom controls, and want to build their own but still utilize the feature of using subtitles for assets that has them, this is method of accessing the subtitle info for the currently loaded asset.
player.getVersion() - Returns the player version
player.getName() - Returns the player name, like Free Video Player
player.isModule() - Checks if the player is a module to itself, which should return false. Free Video Player is built as a module system with each module responsible for each specific tasks, so all modules contain this public method to decern if the instance is a module to Free Video Player.
