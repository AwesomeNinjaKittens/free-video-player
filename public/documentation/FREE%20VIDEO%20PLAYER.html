FREE VIDEO PLAYER

A cool FREE VIDEO PLAYER library to use when want to play DASHed content, Requires the xml2json library to work. Check out more @ FreeVideoPlayer.org

Methods

<static> load(videoUrl, optionalConfigurationObject)

This is the main load method, this method parses the video url and based on that decides which format the video is. If its an adaptive bitrate stream or a regular stream like mp4 for instance.
Parameters:
Name Type Description
videoUrl string The video url, ending with things like .mp4, .webm or .mpd
optionalConfigurationObject object An optional configuration object
