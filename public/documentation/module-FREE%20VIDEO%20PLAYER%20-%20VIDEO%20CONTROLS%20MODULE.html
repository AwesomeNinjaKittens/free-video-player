Module: FREE VIDEO PLAYER - VIDEO CONTROLS MODULE

A module handling creation and updating of video controls for the Free Video Player, to use with the FREE VIDEO PLAYER library. Check out more @ FreeVideoPlayer.org

<inner> updateProgressBarWithBufferedData(bufferedStart, bufferedEnd, totalDurationInSeconds)

A public method accessible to the adaptive streaming module, so the adaptive streaming methods or a regular load non adaptive content method (from the Free Video Player main class), can fire and update how much buffering of the progress bar has been done.
Parameters:
Name Type Description
bufferedStart number When buffering started (usually at 0, for VOD etc)
bufferedEnd number How much has been buffered so far
totalDurationInSeconds string | number The total duration of the media
